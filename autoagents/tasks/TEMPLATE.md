# FEAT task template

Copy structure when creating FEAT files manually (001 agent usually creates these).

```markdown
# <title from issue>

## GitHub Issue
- **Issue:** https://github.com/AMVARA-CONSULTING/km0-web/issues/<N>
- **Number:** #<N>

## Problem / goal
<condensed from issue>

## High-level instructions for coder
- <bullets — no code patches>
- Reference docs/runbook.md when ops-related

## Testing instructions
(to be filled by coder before UNTESTED-)
```

See **`templates/FEAT_TEMPLATE.md`** for a fuller scaffold.

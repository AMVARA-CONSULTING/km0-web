<section class="doc-lead-block">
  <p class="doc-block-title">Einführung</p>
  <p class="doc-lead">Diese Anleitung erklärt, wie Sie die OpenCloud-App auf Android herunterladen, mit KM0 Cloud verbinden und sich anmelden.</p>
</section>

<section class="doc-block">
  <p class="doc-block-title">Schritt 1</p>
  <h2 class="doc-block-heading">App herunterladen</h2>
  <ol class="doc-list">
    <li>Öffnen Sie den Google Play Store auf Ihrem Android-Gerät.</li>
    <li>Suchen Sie nach <strong>OpenCloud</strong>.</li>
    <li>Laden Sie die offizielle App herunter und installieren Sie sie.</li>
  </ol>
  <p class="doc-block-intro">Direkter Link: <a href="https://play.google.com/store/apps/details?id=eu.opencloud.android" target="_blank" rel="noopener noreferrer">OpenCloud im Google Play Store</a>.</p>
</section>

<section class="doc-block doc-block-alt">
  <p class="doc-block-title">Schritt 2</p>
  <h2 class="doc-block-heading">Server beim ersten Start einrichten</h2>
  <ol class="doc-list">
    <li>Öffnen Sie OpenCloud zum ersten Mal.</li>
    <li>Geben Sie bei Aufforderung die OpenCloud-Serveradresse ein.</li>
    <li>Tragen Sie dieselbe URL ein wie im Browser: <code>https://cloud.km0digital.com/</code>.</li>
    <li>Wenn die App eine Kopierfunktion für die URL anbietet, nutzen Sie sie, um Tippfehler zu vermeiden.</li>
  </ol>
</section>

<section class="doc-block">
  <p class="doc-block-title">Schritt 3</p>
  <h2 class="doc-block-heading">Weiter zur Anmeldung</h2>
  <ol class="doc-list">
    <li>Nach der URL-Eingabe tippen Sie auf <strong>Weiter</strong> oder das Pfeilsymbol (je nach App-Version).</li>
    <li>Sie sehen den Anmeldebildschirm mit denselben Optionen wie im Web: eigenes Konto, Google, Apple oder andere, je nach Serverkonfiguration.</li>
  </ol>
  <p class="doc-block-intro">Mehr Details zur Anmeldung finden Sie im Tutorial <a href="/de/tutorials/getting-started-web/">Erste Schritte im Browser</a>.</p>
</section>

<section class="doc-block doc-block-alt">
  <p class="doc-block-title">Häufige Probleme</p>
  <h2 class="doc-block-heading">Fehlerbehebung (Android und iOS)</h2>
  <p class="doc-block-intro"><strong>Fehler: "Wrong server address format" (falsches Serveradressformat)</strong></p>
  <ul class="doc-list">
    <li>Prüfen Sie, ob die URL genau die von KM0 Cloud vorgegebene ist: <code>https://cloud.km0digital.com/</code>.</li>
    <li>Stellen Sie sicher, dass am Anfang oder Ende keine Leerzeichen stehen.</li>
    <li>Lassen Sie <code>https://</code> nicht weg und verwenden Sie keine falschen Zeichen.</li>
  </ul>
  <p class="doc-block-intro">Wenn das Problem bestehen bleibt, wenden Sie sich über die offiziellen Kanäle an den KM0-Support.</p>
</section>

<section class="doc-lead-block">
  <p class="doc-block-title">Einführung</p>
  <p class="doc-lead">Diese Anleitung zeigt, wie Sie <strong>KM0 Cloud</strong> unter macOS als Web-App installieren, damit sie wie eine Desktop-Anwendung aus dem Dock oder dem Ordner Programme startet. Kein App-Store-Download nötig.</p>
</section>

<section class="doc-block">
  <p class="doc-block-title">Schritt 1</p>
  <h2 class="doc-block-heading">Installation über Safari (Zum Dock hinzufügen)</h2>
  <ol class="doc-list">
    <li>Öffnen Sie <strong>Safari</strong> auf Ihrem Mac.</li>
    <li>Rufen Sie <a href="https://cloud.km0digital.com/">https://cloud.km0digital.com/</a> auf und melden Sie sich bei Bedarf an.</li>
    <li>Wählen Sie in der Menüleiste <strong>Ablage → Zum Dock hinzufügen…</strong> (macOS Sonoma oder neuer) oder <strong>Ablage → Zum Dock hinzufügen</strong>.</li>
    <li>Bestätigen Sie den Namen (z. B. <strong>KM0 Cloud</strong>) und klicken Sie auf <strong>Hinzufügen</strong>.</li>
  </ol>
  <p class="doc-block-intro">Apple-Dokumentation: <a href="https://support.apple.com/guide/safari/add-to-dock-ibrw9e991864/mac" target="_blank" rel="noopener noreferrer">Website zum Dock in Safari hinzufügen</a> und <a href="https://support.apple.com/en-us/104996" target="_blank" rel="noopener noreferrer">Safari-Web-Apps auf dem Mac verwenden</a>.</p>
</section>

<section class="doc-block doc-block-alt">
  <p class="doc-block-title">Schritt 2</p>
  <h2 class="doc-block-heading">Installation über Chrome, Edge oder Brave (PWA)</h2>
  <ol class="doc-list">
    <li>Öffnen Sie einen Chromium-Browser (Chrome, Microsoft Edge, Brave oder ähnlich).</li>
    <li>Rufen Sie <a href="https://cloud.km0digital.com/">https://cloud.km0digital.com/</a> auf.</li>
    <li>Suchen Sie das Installationssymbol in der Adressleiste (oft ein Monitor mit Pfeil) oder öffnen Sie das Browsermenü und wählen Sie <strong>KM0 Cloud installieren</strong> oder <strong>App installieren</strong>.</li>
    <li>Bestätigen Sie die Installation, wenn Sie dazu aufgefordert werden.</li>
  </ol>
  <p class="doc-block-intro">PWA-Hilfe: <a href="https://developer.mozilla.org/en-US/docs/Web/Progressive_web_apps/Guides/Making_PWAs_installable" target="_blank" rel="noopener noreferrer">MDN: Making PWAs installable</a> und <a href="https://web.dev/learn/pwa/installation/" target="_blank" rel="noopener noreferrer">web.dev PWA installation</a>.</p>
</section>

<section class="doc-block">
  <p class="doc-block-title">Schritt 3</p>
  <h2 class="doc-block-heading">Wo die App erscheint</h2>
  <ul class="doc-list">
    <li><strong>Safari-Web-App:</strong> ein neues Symbol im <strong>Dock</strong>. Auch unter <strong>Programme</strong> im Finder oder über Spotlight auffindbar.</li>
    <li><strong>Chromium-PWA:</strong> meist im Ordner <strong>Programme</strong> und optional im Dock angeheftet.</li>
    <li>Beim Start öffnet sich KM0 Cloud in einem eigenen Fenster, getrennt von normalen Browser-Tabs.</li>
  </ul>
</section>

<section class="doc-block doc-block-alt">
  <p class="doc-block-title">Schritt 4</p>
  <h2 class="doc-block-heading">Installierte App entfernen oder verwalten</h2>
  <ul class="doc-list">
    <li><strong>Safari-Web-App:</strong> Rechtsklick auf das Dock-Symbol → <strong>Optionen → Aus dem Dock entfernen</strong>. Zum Löschen der App-Datei öffnen Sie <strong>Programme</strong> im Finder, suchen die Web-App und verschieben sie in den Papierkorb.</li>
    <li><strong>Chromium-PWA:</strong> in Chrome <code>chrome://apps</code> öffnen, Rechtsklick auf die App → <strong>Aus Chrome entfernen</strong>, oder aus dem Ordner Programme im Finder löschen.</li>
    <li>Das Entfernen der Web-App löscht weder Ihr KM0-Cloud-Konto noch Ihre Dateien.</li>
  </ul>
</section>

<section class="doc-block">
  <p class="doc-block-title">Fehlerbehebung</p>
  <h2 class="doc-block-heading">Installationsoption nicht sichtbar</h2>
  <ul class="doc-list">
    <li><strong>Safari:</strong> Web-Apps erfordern macOS Ventura 14 oder neuer. Aktualisieren Sie macOS und Safari und laden Sie <a href="https://cloud.km0digital.com/">cloud.km0digital.com</a> neu.</li>
    <li><strong>Chromium:</strong> der Installationshinweis erscheint manchmal erst nach mehreren Besuchen oder nach der Anmeldung. Versuchen Sie das Browsermenü → <strong>Übertragen, speichern und teilen</strong> → <strong>Seite als App installieren</strong> (Formulierung je nach Browser unterschiedlich).</li>
    <li><strong>Privates Surfen:</strong> Installation ist in privaten Fenstern oft deaktiviert. Verwenden Sie ein normales Fenster.</li>
    <li><strong>Firmen-Mac:</strong> Ihre Organisation kann Web-Apps einschränken. Fragen Sie die IT-Abteilung oder nutzen Sie den Browser.</li>
  </ul>
  <p class="doc-block-intro">Sie können KM0 Cloud jederzeit in einem normalen Browser-Tab nutzen. Siehe <a href="/de/tutorials/getting-started-web/">Erste Schritte im Browser</a>.</p>
</section>

<section class="doc-lead-block">
  <p class="doc-block-title">Einführung</p>
  <p class="doc-lead">Diese Anleitung erklärt, wie Sie KM0 Cloud von null an in einem Webbrowser nutzen. Sie brauchen keine technischen Kenntnisse, nur Ihren üblichen Browser und Ihre Zugangsdaten.</p>
</section>

<section class="doc-block">
  <p class="doc-block-title">Schritt 1</p>
  <h2 class="doc-block-heading">Öffnen Sie die KM0-Cloud-URL</h2>
  <ol class="doc-list">
    <li>Öffnen Sie Ihren Browser (Chrome, Firefox, Safari, Edge oder einen anderen).</li>
    <li>Gehen Sie zu <a href="https://cloud.km0digital.com/">https://cloud.km0digital.com/</a>.</li>
    <li>Wenn Sie noch nicht angemeldet sind, fordert die Seite Sie zur Anmeldung auf.</li>
  </ol>
</section>

<section class="doc-block doc-block-alt">
  <p class="doc-block-title">Schritt 2</p>
  <h2 class="doc-block-heading">Anmeldeoptionen</h2>
  <p class="doc-block-intro">Sie können sich auf verschiedene Weise anmelden:</p>
  <ul class="doc-list">
    <li><strong>KM0-Cloud-Konto:</strong> E-Mail oder Benutzername und Passwort.</li>
    <li><strong>Externe Anbieter:</strong> Google oder Apple, falls auf dem Server aktiviert.</li>
    <li><strong>Neue Registrierung:</strong> wenn Sie noch kein Konto haben, können Sie sich auf derselben Anmeldeseite registrieren.</li>
  </ul>
</section>

<section class="doc-block">
  <p class="doc-block-title">Schritt 3</p>
  <h2 class="doc-block-heading">Anmeldung mit Google oder Apple</h2>
  <ol class="doc-list">
    <li>Tippen Sie auf die Google- oder Apple-Schaltfläche auf dem Anmeldebildschirm.</li>
    <li>Wählen Sie das Konto, das Sie verwenden möchten.</li>
    <li>Bestätigen Sie Berechtigungen, falls Ihr Browser danach fragt.</li>
  </ol>
  <p class="doc-block-intro">Der Vorgang ist meist in wenigen Sekunden abgeschlossen.</p>
</section>

<section class="doc-block doc-block-alt">
  <p class="doc-block-title">Schritt 4</p>
  <h2 class="doc-block-heading">Registrierung mit eigenem Konto</h2>
  <ol class="doc-list">
    <li>Wählen Sie die Registrierungsoption auf der Anmeldeseite.</li>
    <li>Geben Sie Ihre E-Mail-Adresse ein.</li>
    <li>Wählen Sie einen Benutzernamen.</li>
    <li>Legen Sie ein sicheres Passwort fest, wie auf jeder anderen Website.</li>
  </ol>
</section>

<section class="doc-block">
  <p class="doc-block-title">Schritt 5</p>
  <h2 class="doc-block-heading">Nach der Anmeldung</h2>
  <p class="doc-block-intro">Nach der Anmeldung werden Sie in Ihren Cloud-Bereich weitergeleitet. Von dort können Sie Dateien hochladen, teilen und mit Ihrem Team zusammenarbeiten.</p>
</section>

<section class="doc-block doc-block-alt">
  <p class="doc-block-title">Häufige Probleme</p>
  <h2 class="doc-block-heading">Fehlerbehebung (Web)</h2>
  <ul class="doc-list">
    <li><strong>Passwort vergessen:</strong> nutzen Sie die Wiederherstellungsoption auf der Anmeldeseite.</li>
    <li><strong>Google- oder Apple-Schaltfläche fehlt:</strong> diese Methode ist auf Ihrem Server möglicherweise nicht aktiviert; versuchen Sie Ihr KM0-Cloud-Konto.</li>
    <li><strong>Seite lädt nicht:</strong> prüfen Sie Ihre Internetverbindung und ob die URL genau <code>https://cloud.km0digital.com/</code> lautet.</li>
  </ul>
  <p class="doc-block-intro">Bei Serveradressfehlern in der mobilen App siehe die Tutorials für <a href="/de/tutorials/getting-started-android/">Android</a> oder <a href="/de/tutorials/getting-started-ios/">iOS</a>.</p>
</section>

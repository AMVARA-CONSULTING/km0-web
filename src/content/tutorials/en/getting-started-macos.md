<section class="doc-lead-block">
  <p class="doc-block-title">Introduction</p>
  <p class="doc-lead">This guide shows how to install <strong>KM0 Cloud</strong> as a web app on macOS so it opens like a desktop application from the Dock or Applications folder. No App Store download is required.</p>
</section>

<section class="doc-block">
  <p class="doc-block-title">Step 1</p>
  <h2 class="doc-block-heading">Install from Safari (Add to Dock)</h2>
  <ol class="doc-list">
    <li>Open <strong>Safari</strong> on your Mac.</li>
    <li>Go to <a href="https://cloud.km0digital.com/">https://cloud.km0digital.com/</a> and sign in if needed.</li>
    <li>In the menu bar, choose <strong>File → Add to Dock…</strong> (macOS Sonoma or later) or <strong>File → Add to Dock</strong>.</li>
    <li>Confirm the name (for example <strong>KM0 Cloud</strong>) and click <strong>Add</strong>.</li>
  </ol>
  <p class="doc-block-intro">Apple documentation: <a href="https://support.apple.com/guide/safari/add-to-dock-ibrw9e991864/mac" target="_blank" rel="noopener noreferrer">Add a website to the Dock in Safari</a> and <a href="https://support.apple.com/en-us/104996" target="_blank" rel="noopener noreferrer">Use Safari web apps on Mac</a>.</p>
</section>

<section class="doc-block doc-block-alt">
  <p class="doc-block-title">Step 2</p>
  <h2 class="doc-block-heading">Install from Chrome, Edge, or Brave (PWA)</h2>
  <ol class="doc-list">
    <li>Open a Chromium-based browser (Chrome, Microsoft Edge, Brave, or similar).</li>
    <li>Go to <a href="https://cloud.km0digital.com/">https://cloud.km0digital.com/</a>.</li>
    <li>Look for an install icon in the address bar (often a monitor with a down arrow) or open the browser menu and choose <strong>Install KM0 Cloud</strong> or <strong>Install app</strong>.</li>
    <li>Confirm the installation when prompted.</li>
  </ol>
  <p class="doc-block-intro">General PWA guidance: <a href="https://developer.mozilla.org/en-US/docs/Web/Progressive_web_apps/Guides/Making_PWAs_installable" target="_blank" rel="noopener noreferrer">MDN: Making PWAs installable</a> and <a href="https://web.dev/learn/pwa/installation/" target="_blank" rel="noopener noreferrer">web.dev PWA installation</a>.</p>
</section>

<section class="doc-block">
  <p class="doc-block-title">Step 3</p>
  <h2 class="doc-block-heading">Where the app appears</h2>
  <ul class="doc-list">
    <li><strong>Safari web app:</strong> a new icon in the <strong>Dock</strong>. You can also find it under <strong>Applications</strong> in Finder or via Spotlight search.</li>
    <li><strong>Chromium PWA:</strong> usually in the <strong>Applications</strong> folder and optionally pinned to the Dock.</li>
    <li>Launching the icon opens KM0 Cloud in its own window, separate from regular browser tabs.</li>
  </ul>
</section>

<section class="doc-block doc-block-alt">
  <p class="doc-block-title">Step 4</p>
  <h2 class="doc-block-heading">Remove or manage the installed app</h2>
  <ul class="doc-list">
    <li><strong>Safari web app:</strong> right-click the Dock icon → <strong>Options → Remove from Dock</strong>. To delete the app file, open <strong>Applications</strong> in Finder, locate the web app, and move it to Trash.</li>
    <li><strong>Chromium PWA:</strong> in Chrome go to <code>chrome://apps</code>, right-click the app → <strong>Remove from Chrome</strong>, or delete it from the Applications folder in Finder.</li>
    <li>Removing the web app does not delete your KM0 Cloud account or files.</li>
  </ul>
</section>

<section class="doc-block">
  <p class="doc-block-title">Troubleshooting</p>
  <h2 class="doc-block-heading">Install option not visible</h2>
  <ul class="doc-list">
    <li><strong>Safari:</strong> you need macOS Ventura 14 or later for web apps. Update macOS and Safari, then reload <a href="https://cloud.km0digital.com/">cloud.km0digital.com</a>.</li>
    <li><strong>Chromium:</strong> the install prompt may appear only after you visit the site a few times or sign in. Try the browser menu → <strong>Cast, save, and share</strong> → <strong>Install page as app</strong> (wording varies by browser).</li>
    <li><strong>Private browsing:</strong> install options are often disabled in private or incognito windows. Use a normal window.</li>
    <li><strong>Corporate Mac:</strong> your organisation may restrict web apps. Ask IT or use the browser instead.</li>
  </ul>
  <p class="doc-block-intro">You can always use KM0 Cloud in a regular browser tab. See the <a href="/en/tutorials/getting-started-web/">Getting started in your browser</a> tutorial.</p>
</section>
